## V1 services

The Qredo API v1 provides two services that have different requirements for use:

- [Web3 API Wallet](/web3-wallet)

    Supporting broadcast-ready transaction creation, approval, signing, and status updates.

- [Read-only](/read-only)

    Reporting service.
